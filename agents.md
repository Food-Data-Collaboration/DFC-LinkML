
## Source of truth

The Ontology defines the overall model the DFC operates using
The latest version can be found at:
https://w3id.org/dfc/ontology/src/DFC_BusinessOntology.owl

Reference data and enumerated values are held in SKOS taxonomies
The latest versions can be found here:

https://w3id.org/dfc/taxonomies/facets.rdf
https://w3id.org/dfc/taxonomies/productTypes.rdf
https://w3id.org/dfc/taxonomies/vocabulary.rdf
https://w3id.org/dfc/taxonomies/measures.rdf
https://w3id.org/dfc/taxonomies/facets.rdf
https://w3id.org/dfc/taxonomies/scopes.rdf

To access a specific release version of the Ontology, use the following formula:

"https://w3id.org/dfc/ontology/@"{full version number, e.g. v1.16.0}"/src/DFC_BusinessOntology.owl"
